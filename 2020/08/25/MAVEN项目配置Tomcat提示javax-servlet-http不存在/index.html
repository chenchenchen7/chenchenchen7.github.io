MAVEN项目配置Tomcat提示javax.servlet.http不存在
发表于2020-08-25|更新于2020-08-25
|阅读量:

idea下编译maven项目报错，提示如下

程序包javax.servlet不存在
程序包javax.servlet.http不存在

解决javax.servlet/javax.servlet.http不存在的错误步骤如下

1、选中项目右键单击【Open Module Settings】(在项目文件夹上右键)

2、在弹出的【Project Structure】对话框中切换到【dependencies】选项卡，然后单击右侧的“+”号选中【Library…】

3、在弹出的【choose libraries】对话框中选中‘Tomcat 7.0.57’后单击【Add Selected】
